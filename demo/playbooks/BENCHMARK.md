# Xerj hot-path benchmark

Single node, `--insecure`, release binary. Generated by `demo/playbooks/bench.mjs`.

## Bulk ingest

- Docs ingested: **40,000** (real LLM-telemetry corpus, 10,000/batch)
- Wall time (bulk only): **1.5 s**
- Throughput: **26,529 docs/s**
