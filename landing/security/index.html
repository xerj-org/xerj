SBOM
Every release ships a CycloneDX SBOM alongside the binary. Every Rust dependency, every transitive, every crate version. Auditors verify before deploy.

Build provenance attestations · reproducible builds · cryptographically-signed artifact manifests. SLSA Level 3 target. Roadmap Q3 2026.
SIGNED ARTIFACTS
Signed Git tags · signed release archives · cosign-compatible signatures. Offline verification works with cosign verify-blob.

Same source + same toolchain + same flags = byte-identical binary. Anyone can rebuild and verify.
